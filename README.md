# TodoApp

実行サイト : https://xron0s.github.io/TodoList_demo/

React, TypeScriptを用いて制作した簡易的なtodoアプリです。

#### 使い方

1 '追加するタスク'の欄にタスク名を入力
2 タスクの期日と時間を入力（入力しないことも可能）
3 優先度を'high', 'medium', 'low'から選び追加ボタンを押す

A 並び替えの順番を'追加順', '優先度', '完了', '日付順'から選択することでタスクの並び替えができる
B リスト上でタスクの左側にあるチェックボックスの状態を切り替えることで'完了', '未完了'の状態を切り替え,ソート順に影響を及ぼすことができる
C 'カレンダー表示に切り替え', 'リスト表示に切り替え'ボタンを押すことでタスクの表示状態を切り替えることができ,期日を設定したタスクはカレンダー上で日付を選択することで表示することができる
